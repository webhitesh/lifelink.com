# lifelink.com# LifeLink - Blood Donation Platform

## 📌 Project Overview
LifeLink is a web platform where users can register as donors or requesters and find matching blood donors during emergencies.

## 🚀 Features
- Donor Registration & Login
- Requester Registration & Login
- Matching Blood Donors
- Dashboard System
- Search by blood group & city

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML, CSS

## ⚙️ Setup Instructions
1. Download or clone the project
2. Open XAMPP
3. Start Apache & MySQL
4. Open phpMyAdmin
5. Create a database named: web
6. Import the file: web.sql
7. Place project folder inside htdocs
8. Run: http://localhost/lifelink

## 📂 Database Info
Database Name: web
